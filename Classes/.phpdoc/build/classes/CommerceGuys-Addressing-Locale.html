Gets locale candidates.

public static getCandidates(string $locale[, string|null $fallbackLocale = null ]) : array<string|int, mixed>

For example, "bs-Cyrl-BA" has the following candidates:

  1. bs-Cyrl-BA
  2. bs-Cyrl
  3. bs

The locale is de-aliased, e.g. the candidates for "sh" are:

  1. sr-Latn
  2. sr
